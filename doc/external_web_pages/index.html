Ensj Healthcheck

The EnsEMBL Healthcheck framework is distributed in several forms:

  • As a tar.gz file ensj-healthcheck.tar.gz
  • Java classes ensj-healthcheck.jar
  • Java source ensj-healthcheck-src.jar
  • Javadoc ensj-healthcheck-javadoc.tar.gz or browse
  • Or check it out of CVS; the module name is ensj-healthcheck

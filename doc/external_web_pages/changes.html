Ensj HealthCheck Changes

Version 1.2, 2nd May 2002

  • Many new tests. An up-to-date list of tests can be found here.
  • Integration with the EnsJ EnsEMBL Java API - provides the ability to write tests using the functionality provided by EnsJ.
  • Added several new methods to the base test class; e.g. tests can now have a description field.
  • Reorganised package structure
  • Various bug fixes, performance improvements and code cleanup.
